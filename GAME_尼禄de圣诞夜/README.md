本小游戏是一个收集 & 蹦蹦跳的游戏。

- 游戏简介（雾）

  今年的圣诞夜，立志要成为一名合格圣诞老人的尼禄亲，要收集尽可能多的礼物！帮助尼禄亲在倒计时内，收集从天而降的美食和礼物吧——越多越好哦。

- 游玩方法
  - 鼠标左右移动：控制尼禄左右移动
  - 鼠标左键：控制尼禄起跳
  - 鼠标右键：控制尼禄攻击
- 可玩内容
  - 普通食物：+1分
  - 攻击礼物盒：蹦出5个食物，+5分
